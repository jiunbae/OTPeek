iPhone용 OTPeek

많은 계정도 언제나 손이 닿는 곳에

자주 쓰는 계정을 즐겨찾기에 두고, 폴더로 정리하고, 필요한 코드를 빠르게 찾으세요.

Mac용 OTPeek

데스크톱에서도 모든 것을 한눈에

즐겨찾기, 폴더, 검색, 빠른 작업으로 데스크톱에서도 모든 계정을 편하게 관리하세요.

Windows용 OTPeek

자주 쓰는 코드를 클릭 한 번으로

앱에서 암호화 볼트를 관리하고, 크기에 맞춰 표시되는 Windows 위젯에서 즐겨찾기 코드를 바로 복사하세요.

대부분의 인증 앱은 번거롭습니다

잠금 해제하고, 스크롤하고, 눈을 찌푸리고, 탭. OTPeek은 보안을 조금도 포기하지 않고 코드를 한눈에 보여줍니다.

⚡️

한눈에 보는 코드

홈 화면·데스크톱 위젯이 카운트다운과 함께 실시간 코드를 보여줍니다 — 앱을 열 일이 거의 없습니다.

📋

한 번 탭으로 복사

목록, 위젯, 메뉴 막대, 시스템 트레이 어디서든 코드를 탭하면 클립보드에 복사됩니다.

🔐

암호화 볼트

시크릿은 마스터 암호(Argon2id)로 보호되는 AES-256-GCM 볼트에 저장됩니다. Face ID / Touch ID 잠금 내장.

☁️

프라이빗 동기화

iPhone과 Mac 간 iCloud 동기화는 암호화된 볼트만 옮깁니다 — 코드와 시크릿은 평문으로 기기를 떠나지 않습니다.

🗂

많아도 깔끔하게

서비스 로고, 폴더, 즐겨찾기, 즉시 검색으로 많은 계정도 한눈에 정리됩니다.

🦀

하나의 Rust 코어

iOS, macOS, Windows가 동일한 검증된 Rust 엔진을 공유합니다 — 어디서나 같은 암호화와 백업 형식, 완전한 오픈소스.

무료, 오픈소스, 종속 없음

모든 코드가 공개되어 있고, 데이터는 언제든 공개된 형식으로 내보낼 수 있습니다. OTPeek이 홈 화면에 자리를 얻었다면, 별점이나 리뷰가 큰 힘이 됩니다.
