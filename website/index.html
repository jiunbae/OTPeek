OTPeek

Your codes, at a peek.

A fast, private two-factor authentication app for iPhone, Mac, and Windows. Codes on your home screen, one-tap copy, and end-to-end encrypted sync — built on a shared Rust core.

App Store Windows GitHub

Free & open source · iOS 17+ / macOS 14+ / Windows 11

Why OTPeek?

Most authenticator apps make you unlock, scroll, and squint. OTPeek puts your codes one glance away — without giving up security.

⚡️

Codes at a glance

Home-screen and desktop widgets show live codes with a countdown — no need to even open the app.

📋

One-tap copy

Tap a code anywhere — the list, a widget, the menu bar, the system tray — and it's on your clipboard.

🔐

Encrypted vault

Secrets live in an AES-256-GCM vault protected by your master password (Argon2id). Face ID / Touch ID app lock built in.

☁️

Private sync

iCloud sync between iPhone and Mac moves only the encrypted vault — codes and secrets never leave your device in the clear.

🖼

Recognize at a glance

Real service logos on every account, with folders, favorites, and instant search to keep big collections tidy.

🦀

One Rust core

iOS, macOS, and Windows share the same audited Rust engine — identical crypto and backup format everywhere, fully open source.

Designed for speed

The same clean experience on every screen you own.

Free, open source, no lock-in

Every line of code is public. Your data exports to an open, documented format any time. If OTPeek saves you a few seconds every day, you can support its development.
